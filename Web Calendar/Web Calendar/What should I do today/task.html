Description

These days, our lives are very busy and eventful. It's very hard to keep track of everything that happens around us. Let's create a web calendar that will store and manage all upcoming events. Use the Flask framework and create a REST API to save and manage events. The Flask framework allows you to start a web application from a single Python file.

Theory

Create a Flask application object to start working with it:

from flask import Flask

app = Flask(__name__)

To create REST endpoints, use the Flask-RESTful extension. It allows you to create API endpoints and handle incoming requests from users. But first, you need to create an API object that accepts a flask application object as an argument:

from flask_restful import Api, Resource

...

api = Api(app)

After that, you need to create a class that extends the Resource class from the flask_restful module:

class HelloWorldResource(Resource):
    def get(self):
        return {"message": "Hello from the REST API!"}

Create a get() method inside this Resource to parse a GET request to a specific endpoint. To specify an endpoint, call the add_resource method of the API object and pass a resource class and its endpoint:

api.add_resource(HelloWorldResource, '/hello')

After that, all you need to do is to run your application. You can use the following pattern to run a flask application:

if __name__ == '__main__':
    if len(sys.argv) > 1:
        arg_host, arg_port = sys.argv[1].split(':')
        app.run(host=arg_host, port=arg_port)
    else:
        app.run()

That's it! Now, when users send a GET request to the /hello endpoint, they will get the following response:

{
   "message":"Hello from the REST API!"
}

You can learn more about the Flask-RESTful extension in the official Quickstart Guide.
